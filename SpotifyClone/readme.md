=============**Spotify** **Clone**==============



This is a Spotify Clone music player website made using \*\*HTML, CSS and JavaScript\*\*.

It shows playlists (albums) and lets you play songs from different folders.



==============**Features**==============



\- Spotify like UI design

\- Album cards are loaded dynamically

\- Songs list is loaded automatically from selected album

\- Play / Pause music

\- Next / Previous song controls

\- Seekbar (click to change song position)

\- Song time display

\- Volume control

\- Mute / Unmute option

\- Responsive design (Hamburger menu for small screens)



============**Technologies Used**=========

\- HTML

\- CSS

\- JavaScript

===============**How Albums Work**===============



Inside the `songs/` folder, each album should have:

\- `cover.jpg` (album image)

\- `info.json` (album title and description)

\- `.mp3` files (songs)



Example:



songs/

└── ncs/

├── cover.jpg

├── info.json

├── song1.mp3

└── song2.mp3

===============================================



==============**How to Run This Project** =============

Best way to run:



1\. Open the project in VS Code

2\. Install Live Server Extension

3\. Right click on `index.html`

4\. Click \*\*Open with Live Server\*\*



Now the project will run in your browser.



=========Remark ========

This project uses `fetch()` to load songs, so it must be opened using a local server (Live Server).

It will not work properly if you open the file directly.
